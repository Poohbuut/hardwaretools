# images/

Place all site image assets in this folder.

## Naming convention

  product-dovetail-saw.jpg
  product-layout-square.jpg
  product-joiners-mallet.jpg
  product-marking-gauge.jpg
  product-driver-set.jpg
  product-chisel-set.jpg
  hero-banner.jpg
  logo.svg

## Usage in HTML

Reference images with a root-relative path from the html/ folder:

  <img src="../images/product-dovetail-saw.jpg" alt="Dovetail Saw">

## Recommended sizes

  Product cards:   800 × 800 px  (1:1, displayed at 200–400 px)
  Hero banner:    1600 × 700 px  (16:7 ratio to match .slide aspect-ratio)
  Thumbnails:      400 × 400 px  (shown at ~120 px)

## Current placeholders

The HTML files currently use emoji as stand-in product images inside
.product-img and .slide-icon elements. Replace those with <img> tags
pointing here once real photography is available.
